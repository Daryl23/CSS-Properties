Responsive Units Comparison

Introduction to Responsive Units

Responsive units allow web pages to adapt to different screen sizes, improving readability and layout flexibility. Below is a table comparing the most commonly used responsive units in CSS:

Unit Description Base Size Example Size View on Screen
px (Pixels) Absolute unit; fixed size regardless of viewport - 12px 12px
em Relative to the font-size of the parent element Parent element font-size (12px) 2em (24px based on 12px parent) 2em
rem Relative to the root (html) element's font-size Root element font-size (12px) 2rem (24px based on 12px root) 2rem
vw (Viewport Width) Percentage of the viewport width 100vw = full width of the viewport 10vw (10% of viewport width) 10vw
vh (Viewport Height) Percentage of the viewport height 100vh = full height of the viewport 10vh (10% of viewport height) 10vh

Resize the window to see the effect of `vw` and `vh` on the size of the boxes. For `em` and `rem`, notice how the sizes differ when relative to the parent or root element.
