Introduction to Color Values

Colors in web design can be defined using various methods. Here are three common ways:

  • RGB (Red, Green, Blue): Defines colors based on the intensity of red, green, and blue light. Values range from 0 to 255. Example: rgb(255, 0, 0) for red.
  • HEX (Hexadecimal): A six-digit code that represents colors. It combines red, green, and blue values in hexadecimal format. Example: #FF0000 for red.
  • HSL (Hue, Saturation, Lightness): Describes colors using hue (color type), saturation (color intensity), and lightness (brightness). Example: hsl(0, 100%, 50%) for red.

Color Functions

Linear gradients create a smooth transition between multiple colors. The direction and spread of the gradient can be controlled with different settings:

  • Direction: Specifies the direction of the gradient, such as `to right`, `to bottom`, `to top left`, or using angles like `45deg`.
  • Color Stops: Define specific colors at different points along the gradient. For example, linear-gradient(to right, #FF0000 0%, #00FF00 100%).
Gradient: Red to Lime (right): linear-gradient(to right, #FF0000, #00FF00)
Gradient: Blue to Yellow (bottom): linear-gradient(to bottom, #0000FF, #FFFF00)
Gradient: Tomato to Steel Blue (45deg): linear-gradient(45deg, #FF6347, #4682B4)
Gradient: Orange Red to Dark Turquoise (top right): linear-gradient(to top right, #FF4500, #00CED1)
Gradient: Deep Pink to Dark Orange (bottom left): linear-gradient(to bottom left, #FF1493, #FF8C00)
