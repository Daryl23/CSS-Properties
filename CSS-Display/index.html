Display: inline - Elements with `inline` display are placed in the same line, like text. Their width and height are dictated by content, and they do not start on a new line.

Display: inline-block - Elements with `inline-block` display behave like inline elements but allow setting width and height. They flow like inline elements but can have dimensions like block elements.

Display: block - Elements with `block` display take up the full width available and always start on a new line. Block elements respect width and height settings.

Display: flex - Elements with `flex` display are part of a flexible container, which arranges its children in a row or column. Flex allows for flexible sizing and alignment of items.

Display: grid - Elements with `grid` display are part of a grid container, which arranges its children into rows and columns, allowing complex layouts with precise control over placement.
